# Dramagic ✨

一个现代化的 AI 视频生成平台，采用画布式节点编辑器，支持 Sora 2 文生视频。

## ✨ 最新功能

🔐 **密码保护** - 访问控制和安全管理
- 🔑 登录页面 - 精美的登录界面
- 🛡️ 后端验证 - 密码保存在服务器
- 💾 状态保持 - 刷新页面保持登录
- 🚪 退出功能 - 一键退出登录
- 🔒 默认密码：`Dramagic2026`

🎨 **画布编辑器** - 全新的节点式编辑界面
- 📝 文本节点 - 添加和编辑文本
- 🎬 Sora 2 视频节点 - AI 文生视频
- 💰 按视频长度计费 - $0.1/秒视频
- 🔄 自动轮询 - 200秒后开始，每5秒轮询
- ⏱️ 倒计时进度条 - 200秒可视化倒计时
- 🚀 拖拽性能优化 - 流畅跟手
- 📐 视频尺寸限制 - 不会撑大节点
- 🔵🟢🟠 按钮状态 - 智能显示和颜色区分
- 🎨 纯白风格设计

## 项目结构

```
dramagic/
├── frontend/          # Vue 3 前端（部署到 Vercel）
├── backend/           # Node.js 后端（部署到 Railway）
├── docs/              # 项目文档
│   ├── product.md     # 产品文档
│   ├── development.md # 开发文档
│   ├── api.md         # API 文档
│   └── deployment.md  # 部署文档
└── .cursorrules       # 开发规范
```

## 技术栈

### 前端
- Vue 3
- Vite
- 部署平台：Vercel

### 后端
- Node.js
- Express
- 部署平台：Railway

## 快速开始

### 前端开发
```bash
cd frontend
npm install
npm run dev
```

### 后端开发
```bash
cd backend
npm install
npm run dev
```
