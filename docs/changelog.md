# Dramagic 更新日志

> 记录产品版本更新内容

---

## v0.06 (2026-01-29)

### 新功能
- 画布视频节点支持切换生成来源（fal.ai / Comfly 官方优质版 / Comfly Original）
- Playground 卡片改为普通静态卡片（移除拖拽功能）

---

## v0.05 (2026-01-29)

### 新功能
- 新增价格配置模块，支持美元/人民币双币种显示
- 新增价格查询 API 接口
- 支持自定义汇率配置
- 视频生成节点显示真实价格（美元 + 人民币）

---

## v0.04 (2026-01-29)

### 新功能
- 新增项目管理功能（新建/保存/打开/删除项目）
- 新增画布状态云端保存
- 新增多人协同编辑（实时同步）
- 新增在线人数显示
- 新增本地自动保存和断网恢复
- 新增未完成任务自动恢复
- 画布升级为硅谷风格点阵背景
- 节点卡片支持自由缩放
- 新增「查看详情」弹窗功能

### Bug 修复
- 修复视频节点重新生成时错误信息未清除的问题
- 修复 Playground 卡片错误状态共享的问题
